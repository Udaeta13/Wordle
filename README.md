# Wordle

## Creado por Julen Udaeta

1. Conexion a Api para obtener la palabra aleatoria
2. Colores en las letras para ver si la letra es correcta
3. Api que comprueba con la RAE si las palabras existen (In progress)

| Palabra        | Feedback  |
|----------------|-----------|
| Perro          | RARRR     |
| Miedo          | RVARR     |
| Finge          | RVRAV     |
| Ligue          | VVVVV     |
